# Customer Insights & Recommendation Dashboard

Week 2 Project — Data Analytics Internship at ReadyNest Corp

## Tools Used
Python | Pandas | Jupyter Notebook | Power BI | Microsoft Word

## Project Overview
Analyzed 524,878 online retail transactions to uncover customer insights and business recommendations.

## Key Insights
- 274 High Value customers drive 61.5% of total revenue
- 65.6% repeat buyer rate
- Revenue peaks every Q4
- 34.4% one-time buyers represent re-engagement opportunity
